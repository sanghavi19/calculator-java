# Calculator (Java)

A simple command-line calculator built with Java that performs basic arithmetic operations.

## Features
- Addition, Subtraction, Multiplication, Division
- Handles division by zero error
- Easy to use command-line interface
- Type 'exit' to quit the app

## Requirements
- Java JDK 8 or above

## How to Run
1. Clone the repo
2. Compile: javac Calculator.java
3. Run: java Calculator

## Example Usage
Enter expression (e.g. 5 + 3) or 'exit' to quit: 10 / 2
Result: 5.0
